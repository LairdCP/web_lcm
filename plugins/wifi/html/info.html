SSID – Service Set Identifier. Unique name of up to 32 characters that identifies a particular 802.11 WLAN. Client Name – Device name assigned to the Laird radio and the client device that uses it. If CCX Features are set to on or optimized in the Global settings page, then the client name is relayed during association. TX Power – Set the Maximum transmit power of the radio, in milliwatts (mW) for this profile. Auth Type – Sets the 802.11 authentication type used when associating to an access point. Radio Mode – Sets the use of 802.11a, 802.11g, 802.11b, and 802.11n frequencies and data rates when interacting with an access point, or the use of ad hoc mode to join a peer-to-peer network instead of associating to an access point. Power Save – Sets the wireless interface's current power save setting. PSP Delay – Sets the wireless interface's current PSP Delay setting. Security Type – Sets the wireless interface's current encryption and key management setting for the this profile. EAP Type – Sets the wireless interface's EAP setting for this profile. WEP Key Index – Sets the wireless interface's WEP transmit key for this profile. WEP Key – Sets the wireless interface's WEP key for index 1 for this profile. WEP Key – Sets the wireless interface's WEP key for index 2 for this profile. WEP Key – Sets the wireless interface's WEP key for index 3 for this profile. WEP Key – Sets the wireless interface's WEP key for index 4 for this profile. PreShared Key – Sets the wireless interface's PSK, or Pre-Shared keygen for this profile. Username – Sets the wireless interface's username for this profile. Password – Sets the wireless interface's password for this profile. User Cert – Sets a user certificate for this profile. User Cert Password – Sets a password associated with a user certificate for this profile. CA Cert – Sets the root certificate authority digital certificate for this profile. PAC Filename – Sets the Protected Access Credential (PAC) file for this profile. PAC Password – Sets a password associated with a PAC file certificate for this profile. A Channel Set – Sets a specific set of channels to operate on in the 5 GHz band. Valid operating channels are constrained by the configured regulatory domain. Auth Server Type – Indicates the type of authentication server being used for EAP. Auto Profile – Manages the auto profile mode. BG Channel Set – Sets a specific set of channels to operate on in the 2.4 GHz band. Valid operating channels are constrained by the configured regulatory domain. Beacon Miss Time – Indicates the beacon missed time in TUs before considering to switch to a different BSS. BT Coexist – Enables or disables Bluetooth coexistence. CCX features – Enables or disables CCX features. Certificate Path – Used to change the path of the certificate store. Date Check – Validates certificates against the system date and time. Default Adhoc Channel – Indicates the channel to be used for an ad hoc connection if the active profile has a Radio Mode value of Ad Hoc. DFS Channel – Indicates support (or lack of support) for 5 GHz (802.11a) channels where DFS is required. Frag Threshold – If packet size (in bytes) exceeds this threshold, then the packet is fragmented. FIPS – Turns FIPS compatibility on (Supporting FIPS requires additional integration efforts). PMK Caching – Indicates the type of PMK caching to use with a WPA2 encryption type. Probe Delay – Maximum number of seconds to wait between scans during a connection attempt. Regulatory Domain – Shows the current regulatory domain. Roam Period ms – The amount of time between roam scans. Roam scans occur after the radio has fallen below the roam trigger. Roam Trigger – The signal strength (RSSI) (in dBm) at which the radio scans for an access point with a better signal strength. RTS Threshold – The packet size above which RST/CTS is required on link. Scan DFS Time – Indicates the dwell (listen) time when passively scanning on a DFS channel. TTLS Inner Method – Indicates the authentication method that is used within the secure tunnel created by EAP-TTLS. UAPSD – Enables or disables UAPSD features. WMM – Enables or disables WMM features.
